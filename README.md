# SkyproWallet

## О проекте

**SkyproWallet** — это веб-приложение для учета финансовых операций пользователя.
Приложение позволяет добавлять доходы и расходы, анализировать финансовую активность и просматривать статистику.

Проект разрабатывается в рамках командной домашней работы.

---

## Основной функционал

* Регистрация и авторизация пользователя
* Добавление доходов и расходов
* Просмотр таблицы операций
* Анализ финансов с помощью графиков
* Календарь операций
* Адаптивный интерфейс

## Установка и запуск проекта

### 1. Клонировать репозиторий

```bash
git clone https://github.com/your-username/skypro-wallet.git
```

### 2. Перейти в папку проекта

```bash
cd skypro-wallet
```

### 3. Установить зависимости

```bash
npm install
```

### 4. Запустить проект

```bash
npm start
```

После запуска приложение будет доступно по адресу:

```
http://localhost:3000
```

## Работа с Git

### Создание ветки

Перед началом задачи создаётся новая ветка:

```bash
git checkout -b feature/task-name
```

### Pull Request

Каждая задача отправляется через **Pull Request**.

В PR необходимо указать:

* ссылку на задачу из Kaiten
* краткое описание изменений
* плановое и фактическое время выполнения

Пример:

```
PR: Создание компонента кнопки

Создан переиспользуемый компонент Button.
Добавлены состояния active / disabled.
```

## Доска задач
