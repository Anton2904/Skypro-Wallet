# Skypro Wallet

Skypro Wallet — учебное React-приложение для учета личных расходов. Проект позволяет зарегистрироваться, войти в аккаунт, добавлять и удалять транзакции, а также смотреть аналитику расходов по выбранному периоду.

## Основные возможности

- регистрация и авторизация пользователя
- сохранение токена и данных пользователя в `localStorage`
- просмотр списка расходов на главной странице
- добавление новой транзакции через форму
- удаление транзакции из таблицы
- анализ расходов по периодам: день, неделя, месяц
- столбчатая диаграмма с суммой по категориям
- адаптивная верстка для desktop, tablet и mobile
- обработка ошибок форм и базовых пользовательских сценариев

## Стек технологий

```bash

```

Приложение будет доступно в браузере после запуска Vite dev server.

## Структура проекта

- `src/pages` — страницы приложения
- `src/components` — переиспользуемые UI-компоненты
- `src/api` — слой работы с данными
- `src/context` и `src/hooks` — контекст авторизации и пользовательские хуки
- `src/utils` — форматирование, валидация и работа с `localStorage`
- `src/data` — стартовые тестовые данные

## Командная работа

Работа над проектом ведется по kanban-подходу:

- каждая задача имеет описание, оценку и критерии готовности
- все изменения проходят через pull request
- в PR указываются ссылка на задачу, плановое и фактическое время, а также краткое описание выполненной работы
- после завершения задача переносится в колонку «Готово»



### Пример описания pull request

```text
Задача: https://kaiten.ru/...
Плановое время: 3 часа
Фактическое время: 3.5 часа

Что сделано:
- реализована регистрация и авторизация
- добавлен выход из приложения
- реализовано удаление транзакций
- улучшена адаптивность и проведен рефакторинг
```

## Дополнительно

Проект не использует реальный backend: авторизация, регистрация и транзакции реализованы через `localStorage`. Это позволяет продемонстрировать работу пользовательских сценариев и интерфейса в рамках учебного задания.
