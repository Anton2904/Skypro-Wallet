# Skypro Wallet

Учебный проект по React и Vite для учёта личных расходов. В проекте реализована базовая архитектура приложения, стартовая верстка основных экранов и подготовлена структура для дальнейшего подключения бизнес-логики и API.

## Что реализовано

- стартовая структура проекта на React + Vite;
- экраны авторизации и регистрации;
- страница «Мои расходы» с таблицей и формой добавления расхода;
- страница «Анализ расходов» с календарём и диаграммой;
- единый Header для внутренних страниц;
- адаптивная верстка для desktop, tablet и mobile;
- настроены ESLint и Prettier;
- готовая структура для командной разработки по спринтам.

## Технологии

- React
- Vite
- JavaScript (ES6+)
- CSS3
- ESLint
- Prettier

## Структура проекта

```text
src/
  components/   # переиспользуемые UI-компоненты
  data/         # временные мок-данные для верстки
  pages/        # страницы приложения
  App.jsx       # точка маршрутизации экранов
  App.css       # основные стили приложения
  index.css     # глобальные стили
```

## Как запустить проект

```bash
npm install
npm run dev
```

## Дополнительные команды

```bash

```

## Маршруты экранов

- `/login` — страница входа
- `/register` — страница регистрации
- `/expenses` — страница списка расходов
- `/analysis` — страница анализа расходов

## Критерии готовности

- проект устанавливается и запускается без ошибок;
- линтер проходит без предупреждений и ошибок;
- структура проекта понятна и пригодна для командной работы;
- внешний вид соответствует макету;
- верстка адаптируется под разные размеры экрана.
