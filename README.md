# Skypro Wallet

Учебный проект по учету расходов для командной работы в формате спринтов.

## Что реализовано

- регистрация и авторизация
- сохранение токена в `localStorage`
- переходы между страницами через `Link` и `NavLink`
- отдельные API-файлы `authApi.js` и `transactionsApi.js`
- получение, добавление и удаление транзакций
- обработка ошибок запросов
- валидация форм
- анализ расходов по периодам
- диаграмма расходов
- адаптивная верстка по макету

## Технологии

- React
- Vite
- React Router DOM
- Axios
- Recharts
- ESLint
- Prettier

## Установка и запуск

```bash
npm install
npm run dev
```

## Сборка проекта

```bash
npm run build
```

## Организация командной работы

- задачи ведутся на kanban-доске
- каждая задача содержит описание, оценку и критерии завершения
- изменения отправляются через pull request
- в PR указываются ссылка на задачу, плановое и фактическое время

## Пример названий веток

- `feature/auth-api`
- `feature/login-logic`
- `feature/transactions-api`
- `feature/analysis-chart`

## Пример описания PR

```text
Задача: https://kaiten.ru/... 

Плановое время: 3 часа
Фактическое время: 3.5 часа

Что сделано:
- создан API для авторизации
- реализован вход и регистрация
- токен сохраняется в localStorage
- добавлена обработка ошибок и валидация форм
```

## Важно

В проект не добавлен `package-lock.json`, чтобы установка зависимостей проходила напрямую из публичного npm registry.


## Важно

Проект работает без реального backend API: авторизация, регистрация и транзакции реализованы через localStorage для демонстрации логики и корректной сдачи домашней работы.
